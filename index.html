What JSkat is. JSkat is a piece of software written in Java inspired by the German three-player card game named Skat. In functionality, it is similar to the Hearts or Solitaire games found on some Windows machines. The GUI includes a virtual table where cards are dealt. It shows the user's cards face-up, while the faces of opponents' cards are hidden from view. JSkat includes two types of Artificial Intelligence, which guide non-human participants in the game. The project also has a module which provides functionality for playing the game against other human users of the software over the Internet, on official servers.

Project architecture. The project is modularized into several groups of packages that carry similar tasks. The most important folders and subfolders of the root are:

  • ai: short for Artificial Intelligence, controls behaviour of non-human players in a game
    • algorithmic: AI based on simple algorithms
    • nn: contains many classes for managing a neural network, for training, holding data and making informed decisions
    • rnd: implementation of a player that takes random decisions; for testing purposes
  • control: manages the game's mechanics, e.g. assigns roles such as "dealer" to players, cycles the players' turns; in this package there is the master class, which (citing from comments) controls everything in JSkat
    • iss: interface between the local game and the mechanics of the game hosted by the International Skat Server
  • data: manages and provides all the information about the current tables
    • iss: provide data for non-local tables
  • gui: short for Graphical User Interface, contains 91 of the 200 source files; its subfolders also manage different parts of the UI, e.g. chat dialog for ISS games, card faces
    • action: specifies all actions that have to do with the interface, e.g. playing a card, opening the help dialog
  • util: holds information about card roles and ranks, game rules etc.
